# User Flow — 멀티 블로그 수익화 자동화 플랫폼 v2.0

---

## 전체 흐름 한눈에 보기

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│   [1. 블로그 제작]                                                   │
│       회원가입 → 블로그 생성 → 페르소나 설정 → 도메인 연결            │
│       → 디자인 적용 → AdSense 설정 → 블로그 완성!                    │
│                                                                     │
│       ※ 최초 1회 + 새 블로그 추가 시                                 │
│                                                                     │
│   ─────────────────── ↓ ────────────────────                        │
│                                                                     │
│   [2. 키워드 발굴 & 스케줄링]                                        │
│       키워드 발굴(자동+수동) → 등급 분류(S~D) → 블로그 매칭           │
│       → 발행일+시간 배정 → 캘린더 등록                               │
│                                                                     │
│       ※ 자동: 월 1회 계절성 키워드 발굴                              │
│       ※ 수동: 언제든 키워드 탐색                                     │
│                                                                     │
│   ─────────────────── ↓ ────────────────────                        │
│                                                                     │
│   [3. 글 자동 발행]                                                  │
│       예약 시간 도달 → AI 글 생성(PASONA+페르소나)                    │
│       → AI 이미지 생성 → 광고 태그 삽입 → 블로그 발행                 │
│       → 상태 업데이트 → 성과 추적                                    │
│                                                                     │
│       ※ 완전 무인 — 사람 개입 제로                                   │
│                                                                     │
│   ─────────────────── ↓ (루프) ─────────────────                    │
│                                                                     │
│   [피드백 루프] 매주 월요일 자동                                      │
│       실제 수익 vs 예측 비교 → 가중치 자동 보정                       │
│       → 다음 발행에 반영 → 시스템이 스스로 개선됨                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Flow 4. 계절성 키워드 자동 발굴 (월 1회 자동)

```
[매월 1일 00:00 자동 실행]

    Step 1: ANNUAL_EVENTS 캘린더에서 다음달 + 다다음달 이벤트 추출
            예) 현재 3월 → 4월(벚꽃명소, 봄여행지...) + 5월(어버이날선물...) 추출

    Step 2: Claude API → 이벤트별 롱테일 키워드 20개 생성
            "벚꽃 명소 추천" → "서울 벚꽃 명소 2026", "벚꽃 개화시기 예측" ...

    Step 3: 네이버 광고 API + 데이터랩 → 검색량 + YoY 성장률 수집

    Step 4: 5단계 점수 산출 + 등급 분류

    Step 5: keyword_pool 자동 등재 (B등급 — 계절 황금)

    Step 6: 블로그 자동 매칭
            "벚꽃 명소 추천" (travel) → 여행 블로그에 자동 배정
            "어버이날 선물 추천" (shopping) → 가장 적합한 블로그 탐색

    Step 7: 발행일 자동 계산
            피크일(4월 15일) − 21일 = 3월 25일 발행 예정

    Step 8: 발행 시간 랜덤 배정
            3월 25일의 아침/점심/오후 중 빈 시간대에 배정
            이미 그날 2건 예약되어 있다면 → 남은 1슬롯에 배정
            3건 꽉 차 있다면 → 다음 날로 밀리거나 같은 카테고리 다른 블로그로 이동

    Step 9: 캘린더에 자동 등록 (캘린더 뷰 + 표 뷰 모두 반영)

    [사용자 알림]
        → 앱 내 알림: "이번 달 계절성 키워드 23개 발굴됨"
        → 클릭 → 표 뷰로 이동 (B등급 + 이번 달 발굴 필터 적용)
        → 사용자는 필요 시 날짜/블로그/키워드 수정 가능
        → 수정하지 않으면 → 예약 시간에 자동 발행됨
```

## Flow 8. 피드백 루프 (자동 — 매주 월요일)

```
[매주 월요일 03:00 자동 실행]

    Step 1: 지난 7일 모든 발행 글의 actual_revenue 수집 (AdSense API)

    Step 2: Revenue Score 예측값과 비교
            오차율 = |actual - predicted| / predicted

    Step 3: 오차율 > 20%인 키워드 패턴 분석
            예) "법률" 카테고리의 cost 유형 글이 일관되게 예측 초과
            예) "여행" 카테고리의 compare 유형 글이 일관되게 예측 미달

    Step 4: 점수 가중치 자동 보정 (최대 ±10%/주)
            법률+cost: Revenue 가중치 +5% 상향
            여행+compare: Revenue 가중치 -3% 하향

    Step 5: 페르소나별 성과 분석
            김정현 페르소나의 CTR이 이동훈보다 15% 높음
            → 김정현 스타일의 특성(짧은 문장, 질문 도입) 추출
            → 다른 페르소나에 적용 권고

    Step 6: 높은 CTR pasonaType → 다음 발행 비중 자동 조정
            cost 유형의 평균 CTR이 4.5%로 가장 높음
            → 다음 주 스케줄에서 cost 유형 비중 +10%

    Step 7: 낮은 CTR 블로그 → 광고 카테고리 재설정 권고

    [사용자 알림]
        → "이번 주 피드백 루프 완료"
        → 클릭 → 대시보드 > 피드백 상세:
          - 보정된 키워드 12건
          - cost 유형 비중 10% 증가 적용
          - 권고: "보험비교왕 블로그의 여행 글 비중 축소 권고"
          - [적용] [무시] 버튼
```
