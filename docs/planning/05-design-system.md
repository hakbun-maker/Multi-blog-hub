# Multi Blog Hub 디자인 시스템

> 버전: 1.0 | 날짜: 2026-03-04

---

## 1. 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **효율** | 많은 정보를 빠르게 파악하고 행동할 수 있어야 함 |
| **명확** | 여러 블로그가 있어도 어디서 무엇을 하는지 혼란 없이 |
| **신뢰** | 자동화 시스템이 잘 돌아가고 있다는 신뢰감을 줌 |

## 4. 핵심 컴포넌트

### BlogCard (블로그 카드)
- 블로그 이름 + 고유 색상 닷
- 도메인 표시
- 최근 발행 글 수 + 방문자 수
- 빠른 작성 버튼

### PostEditor (글 작성 에디터)
- **AI 모드 탭**: 키워드 입력 → 옵션 설정 → AI 생성
- **직접 작성 탭**: 리치텍스트 에디터 (TipTap)
- **HTML 모드**: 코드 에디터 전환 버튼
- **스니펫 패널**: 우측 사이드 드로어
- **블로그 선택**: 체크박스 멀티 선택

### SchedulerTimeline (타임라인)
- 캘린더 뷰 or 리스트 뷰 토글
- 예약된 발행 항목 표시
- 색상 = 대상 블로그 고유 색상

### StatCard (통계 카드)
- 큰 숫자 + 아이콘
- 증감률 (▲/▼)
- 미니 스파크라인 차트

### SnippetPanel (스니펫 패널)
- 드로어 형태 (에디터 우측)
- 검색 + 카테고리 필터
- 원클릭 삽입

## 6. 레이아웃

### 전체 레이아웃 (앱 셸)
```
┌──────────────────────────────────────────┐
│ 상단 헤더 (로고, 검색, 프로필)             │
├────────┬─────────────────────────────────┤
│ 사이드 │                                 │
│  바     │        메인 콘텐츠              │
│ (240px)│                                 │
│ 대시보드│                                 │
│ 블로그  │                                 │
│ 글작성  │                                 │
│ 스케줄러│                                 │
│ 통계   │                                 │
│ 광고   │                                 │
│ 키워드  │                                 │
│ 설정   │                                 │
└────────┴─────────────────────────────────┘
```

### 반응형 브레이크포인트
```
Mobile:  < 768px   (사이드바 숨김, 하단 탭바)
Tablet:  768-1024px (사이드바 아이콘만)
Desktop: > 1024px  (사이드바 전체)
```
