# User Flow — 멀티 블로그 수익화 자동화 플랫폼

---

## 전체 흐름 개요

```
[최초 진입]
    → 회원가입 / 로그인
    → 블로그 등록 (카테고리 설정)
    → 키워드 탐색 / 자동 발굴
    → 콘텐츠 캘린더 확인
    → AI 글 생성
    → 에디터 검토 & 발행
    → 수익 대시보드 확인
    → (자동) 피드백 루프
```

---

## Flow 1. 온보딩 — 블로그 등록

```
[랜딩 페이지]
    → "무료 시작하기" 클릭
    → 이메일 + 비밀번호 입력 (또는 Google OAuth)
    → 이메일 인증
    → [온보딩 Step 1/3] 블로그 추가
        - 블로그 이름 입력
        - URL 입력
        - 플랫폼 선택 (Tistory / WordPress)
        - Tistory: OAuth 인증 버튼 → 네이버 로그인 → 권한 허용
        - WordPress: 관리자 URL + Application Password 입력
    → [온보딩 Step 2/3] 광고 카테고리 설정
        - 등급표 표시 (S/A/B/C/D 와 CPC 범위)
        - primaryAdCategory 선택 (필수)
        - secondaryAdCategory 선택 (선택)
        - ⚠️ "동일 카테고리 70% 이상 유지 시 고CPC 광고 배정" 안내 표시
    → [온보딩 Step 3/3] AdSense 슬롯 등록
        - slot_top / slot_mid / slot_bottom ID 입력
        - ads.txt 등록 여부 체크
    → 완료 → 대시보드 이동

    [블로그 추가 (기존 사용자)]
        대시보드 → "블로그 추가" → 동일 Step 1~3 반복
```

---

## Flow 2. 키워드 탐색 (수동)

```
[사이드바] "키워드" 메뉴 클릭
    → 키워드 탐색기 화면
    → 검색창에 키워드 입력 (예: "실비보험 비교")
    → "분석" 클릭
    → [로딩 ~3초] 네이버 광고 API + Google KWP + 데이터랩 동시 호출
    → 결과 카드 표시:
        - Traffic Score: 72 / Revenue Score: 85 ★ / Difficulty: 35 / Seasonal Bonus: 0
        - Opportunity Score: 78 → 등급: S (수익 황금)
        - 예상 월 수익: 약 85,000원
        - 연관 키워드 자동 추천 (최대 10개)
    → "키워드 풀에 추가" 클릭
    → 배정 블로그 선택 (primaryAdCategory 일치 블로그 자동 추천)
    → 완료

    [키워드 풀 보기]
        → 등급별 필터 / 블로그별 필터 / 날짜별 필터
        → 클러스터 묶기: 키워드 다수 선택 → "클러스터 생성"
        → 발행일 지정 → 캘린더에 등록
```

---

## Flow 3. 계절성 키워드 자동 발굴 (월 1회 자동)

```
[매월 1일 00:00 자동 실행]
    → ANNUAL_EVENTS 캘린더에서 다음달, 다다음달 이벤트 추출
    → Claude API → 이벤트별 롱테일 키워드 20개 생성
    → 네이버 광고 API + 데이터랩 → 검색량 + YoY 성장률 수집
    → SeasonalBonus > 0 OR YoY > 20% → 필터링
    → keyword_pool 자동 등재 (등급 B — 계절 황금)
    → 발행 최적일 자동 계산 (피크 − 21일)

    [사용자 알림]
        → 앱 내 알림 + (선택) 이메일
        → "이번 달 계절성 키워드 N개 발굴됨 — 확인하기"
        → 클릭 → 키워드 풀 (B등급 필터)
        → 발행 승인 또는 수정 후 캘린더 등록
```

---

## Flow 4. 콘텐츠 캘린더 확인 & 조정

```
[사이드바] "캘린더" 메뉴 클릭
    → 월간 캘린더 뷰 (기본)
        - 날짜별 예정 포스트 수 표시
        - 색상: S=빨강 / B=주황 / A=파랑 / C=회색
    → 포스트 카드 클릭 → 팝업:
        - 메인 키워드 / 등급 / 배정 블로그 / 예상 수익
        - "발행일 변경" (드래그 or 날짜 선택)
        - "블로그 변경" (드롭다운)
        - "AI 생성 시작" 버튼

    [오늘의 발행 큐]
        → 상단 배너: "오늘 발행 예정 N건"
        → 목록 표시 → "전체 자동 발행" 또는 건별 "AI 생성"
```

---

## Flow 5. AI 글 생성 & 에디터

```
[캘린더] 포스트 카드 → "AI 생성 시작"
    OR [키워드 풀] 키워드 선택 → "글 생성"
    → 생성 옵션 화면:
        - 글 유형 선택: compare / solve / cost (AI 자동 추천 + 수동 변경 가능)
        - 목표 자수: 2,000자 (기본)
        - 광고 카테고리: 블로그 primaryAdCategory 자동 로드
    → "생성 시작" 클릭
    → [로딩 ~20초] Claude API 호출
        - PASONA 6단계 구조 자동 적용
        - 섹션 타겟팅 태그 자동 삽입
        - adKeywords S단계 H2에 포함
    → 에디터 화면 이동 (생성 완료)

    [에디터 화면]
        ┌────────────────────────────────────────────────┐
        │ 툴바: B I U H1 H2 | 📢PASONA 🎯타겟ON 🚫무시 🏷️카테고리 │
        ├────────────────────────────────────────────────┤
        │ [회색 배경] P단계: 문제 묘사...                   │
        │ [회색 배경] A단계: 공감 경험담...                 │
        │                                                │
        │ [연두색 배경 — 고CPC 광고 구간]                  │
        │ ## 실비보험 비교 시 꼭 확인할 3가지 (S단계)       │
        │ 보험료 절약 방법, 무료 견적...                    │
        │ ## 더 빠른 방법이 있습니다 (O단계)               │
        │                                                │
        │ [회색 배경] N단계: 긴박감...                     │
        │ [회색 배경] A단계: CTA...                       │
        └────────────────────────────────────────────────┘
        우측 패널:
        - SEO 점수: 메인 키워드 N회 / adKeywords ✅ / 자수 1,950자 ✅
        - adKeywords 힌트: "보험비교", "무료견적" (클릭→복사)
        - 발행 전 체크리스트 (모두 ✅ → 발행 버튼 활성화)

    [발행 버튼 클릭]
        → 검증 통과 여부 확인:
            ✅ section_start 태그 존재
            ✅ S단계 H2에 adKeywords 포함
            ✅ 분량 1,800~2,200자
        → HTML 후처리 자동 실행:
            - slot_mid 광고 슬롯 자동 삽입
            - 헤더/푸터 weight=ignore 처리
        → 블로그 API 전송 (Tistory or WordPress)
        → 성공: "발행 완료!" 토스트 + published_url 저장
        → 실패: 재시도 큐 등록 + 알림
```

---

## Flow 6. 수익 대시보드 확인

```
[사이드바] "대시보드" 메뉴 클릭
    → 상단 요약 카드:
        - 오늘 수익: N원 / 이번 주: N원 / 이번 달: N원
        - 전월 대비 ↑ or ↓ %
    → 블로그별 30일 수익 라인 차트
    → 키워드 성과 TOP 10:
        - 키워드 / 실제 수익 / Revenue Score 예측 / 오차율
    → 예정 발행 현황:
        - 오늘 N건 / 이번 주 N건
        - ⚠️ 계절성 키워드 피크 임박 알림 (D-14, D-7)
    → 시스템 상태:
        - 마지막 계절성 탐색: N일 전
        - API 할당량: 네이버 850/1,000 / 구글 정상
        - 피드백 루프: 지난 월요일 실행 완료
```

---

## Flow 7. 피드백 루프 (자동 — 매주 월요일)

```
[매주 월요일 새벽 자동 실행]
    → 지난 7일 actual_revenue 수집 (AdSense API)
    → Revenue Score 예측값과 비교
    → 오차율 > 20% 키워드 패턴 분석
    → 점수 가중치 ±10% 자동 보정
    → 높은 CTR pasonaType → 다음 발행 비중 자동 조정
    → 낮은 CTR 블로그 → 광고 카테고리 재설정 권고 알림

    [사용자 알림]
        → "이번 주 피드백 루프 완료 — 변경 내역 보기"
        → 클릭 → 대시보드 > 시스템 상태 상세 화면
            - 보정된 키워드 목록
            - 권고 사항: "A 블로그 광고 카테고리를 B→C로 변경 권고"
            - "적용" / "무시" 버튼
```
